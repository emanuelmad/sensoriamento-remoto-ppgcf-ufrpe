# Template de projeto

Use esta pasta como referência para abrir novos estudos de caso.

## Estrutura mínima

- `README.md` com objetivo, área de estudo e dados usados
- `notebook.ipynb` ou `main.py` com o fluxo principal
- `resultados/` para mapas, figuras e tabelas finais
- `dados/` apenas para amostras pequenas ou metadados

## Perguntas que todo projeto deve responder

1. Qual problema geográfico ou ambiental está sendo investigado?
2. Quais sensores e resoluções são adequados?
3. Quais bandas, índices ou classificadores foram usados?
4. Como os resultados serão interpretados no contexto florestal ou ambiental?
