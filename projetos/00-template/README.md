# Template de projeto

Use esta pasta como referencia para abrir novos estudos de caso.

## Estrutura minima

- `README.md` com objetivo, area de estudo e dados usados
- `notebook.ipynb` ou `main.py` com o fluxo principal
- `resultados/` para mapas, figuras e tabelas finais
- `dados/` apenas para amostras pequenas ou metadados

## Perguntas que todo projeto deve responder

1. Qual problema geografico ou ambiental esta sendo investigado?
2. Quais sensores e resolucoes sao adequados?
3. Quais bandas, indices ou classificadores foram usados?
4. Como os resultados serao interpretados no contexto florestal ou ambiental?
