# Documentacao

Esta pasta organiza os documentos que estruturam a disciplina para alem dos slides.

## Arquivos principais

- `cronograma.md`: distribuicao sugerida por semana ou modulo
- `ementa.md`: objetivos, competencias e conteudos
- `metodologia.md`: estrategia didatica e fluxo de aula
- `roteiro-projetos.md`: criterios para orientar os projetos finais
