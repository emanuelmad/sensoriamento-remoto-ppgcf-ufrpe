# Documentação

Esta pasta organiza os documentos que estruturam a disciplina para além dos slides.

## Arquivos principais

- `cronograma.md`: distribuição sugerida por semana ou módulo
- `ementa.md`: objetivos, competências e conteúdos
- `metodologia.md`: estratégia didática e fluxo de aula
- `roteiro-projetos.md`: critérios para orientar os projetos finais
