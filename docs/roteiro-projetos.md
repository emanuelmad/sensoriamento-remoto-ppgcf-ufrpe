# Roteiro de projetos

## Estrutura mínima

Cada projeto deve conter:

- problema ou pergunta de pesquisa
- área de estudo
- fonte dos dados
- justificativa do sensor escolhido
- fluxo de processamento
- resultados e interpretação
- limitações e próximos passos

## Perguntas que devem aparecer no relatório

1. O sensor escolhido responde bem ao problema?
2. Quais bandas, índices ou classificadores foram usados e por quê?
3. Como os resultados dialogam com o contexto ambiental, territorial ou florestal?
4. Quais incertezas ou limitações ainda permanecem?
