# Roteiro de projetos

## Estrutura minima

Cada projeto deve conter:

- problema ou pergunta de pesquisa
- area de estudo
- fonte dos dados
- justificativa do sensor escolhido
- fluxo de processamento
- resultados e interpretacao
- limitacoes e proximos passos

## Perguntas que devem aparecer no relatorio

1. O sensor escolhido responde bem ao problema?
2. Quais bandas, indices ou classificadores foram usados e por que?
3. Como os resultados dialogam com o contexto ambiental, territorial ou florestal?
4. Quais incertezas ou limitacoes ainda permanecem?
